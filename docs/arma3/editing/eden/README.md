## Gruppnamn i lobbyn

I CBA A3 finns en smidig funktion som låter uppdragsskapare sätta namn på sina grupper i lobbyn.
Fyll i rutan "Role Description" med gruppens namn på **samtliga** gruppmedlemmar.
